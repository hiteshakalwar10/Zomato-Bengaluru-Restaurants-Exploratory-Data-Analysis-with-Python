# Zomato Bengaluru Restaurants – Exploratory Data Analysis (EDA)

## Project Overview

This project explores the Zomato Bengaluru Restaurants dataset to uncover meaningful insights using Exploratory Data Analysis (EDA). The goal is to analyze restaurant trends, customer preferences, pricing, and online ordering behavior through data visualization.

This project demonstrates practical data analysis skills using Python, Pandas, Matplotlib, and Seaborn.

## Objectives

- Analyze the most popular cuisines
- Study restaurant rating distribution
- Perform cost analysis
- Compare online and offline ordering availability
- Identify locations with the highest number of restaurants
- Detect correlations and outliers using visualizations
